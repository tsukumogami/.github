# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any tsukumogami project, please
report it responsibly. **Do not open a public issue.**

### How to report

Use GitHub's private vulnerability reporting on the affected repository:

1. Go to the repository's **Security** tab
2. Click **Report a vulnerability**
3. Fill in the details

### What to expect

This is a best-effort personal project. We'll review reports as time allows
and coordinate disclosure with you when possible.

### Scope

This policy applies to all repositories under the
[tsukumogami](https://github.com/tsukumogami) organization.

## Supported Versions

We provide security fixes for the latest release of each project. Older
versions are not actively supported.

| Project | Supported |
|---------|-----------|
| tsuku | Latest release |
| koto | Latest release |
| shirabe | Latest release |
